# ScopeDiff Metrics Dashboard

This is a lightweight manual dashboard for post-release learning. It is not for vanity metrics or growth hacking.

## Current Release

- GitHub: https://github.com/xiwuqi/scopediff
- npm: https://www.npmjs.com/package/scopediff
- Release: https://github.com/xiwuqi/scopediff/releases/tag/v0.1.0

## Weekly Questions

- Can a new user understand the project from the first README screen?
- Does `npx scopediff@latest scan` work for a fresh user?
- Which rules produce useful findings?
- Which rules produce noisy false positives?
- Which community or channel produced real feedback?
- Did any launch action violate the "relevant, transparent, non-spammy" rule?

## Quality Signals

- False-positive reports with sanitized fixtures.
- Issues from users who tried the tool.
- PRs improving fixtures, docs, or rules.
- npm downloads after a specific launch post.
- GitHub stars from organic discovery.

## Anti-Goals

- No bought, traded, automated, or incentivized stars.
- No spam comments or irrelevant community posts.
- No fake testimonials.
- No inflated benchmark claims.
- No posting from the maintainer's accounts without explicit approval.
