# Code of Conduct

ScopeDiff follows the Contributor Covenant Code of Conduct, version 2.1.

## Our Pledge

We pledge to make participation in this project a harassment-free experience for everyone.

## Expected Behavior

- Be respectful and constructive.
- Assume good intent while staying precise.
- Focus criticism on code, docs, behavior, and tradeoffs.
- Report false positives and security concerns responsibly.

## Unacceptable Behavior

- Harassment, threats, or personal attacks.
- Publishing private information.
- Spam, brigading, or manipulative growth tactics.
- Attempts to add hidden telemetry, credential collection, or malicious behavior.

## Enforcement

Maintainers may remove comments, close issues, or block participants who violate these expectations.
